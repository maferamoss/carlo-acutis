# ✝ Santo Carlo Acutis — O Padroeiro da Internet

Site desenvolvido em **ASP.NET Core MVC** em homenagem a Santo Carlo Acutis,
o primeiro santo da geração millennial, canonizado em 7 de setembro de 2025.

## 📖 Sobre o projeto

Carlo Acutis (1991–2006) foi um jovem italiano apaixonado por programação e pela fé católica.
Ele criou um site catalogando milagres eucarísticos de todo o mundo para evangelizar através da internet.
Este projeto recria esse trabalho e conta sua história.

## 📌 Seções do site

- **Biografia** — Quem foi Carlo Acutis
- **Galeria** — Fotos de sua vida
- **Milagres Eucarísticos** — Catálogo de milagres ao redor do mundo
- **Canonização** — A histórica cerimônia de 7 de setembro de 2025
- **Intercessão** — Milagres reconhecidos pela Igreja para sua beatificação e canonização
- **Frases** — Palavras marcantes de Carlo
- **Referências** — Fontes utilizadas

## 🛠️ Tecnologias

- ASP.NET Core MVC
- C#
- HTML5 / CSS3
- Google Fonts (Cinzel + Lora)

## 🚀 Como rodar localmente

```bash
git clone https://github.com/seu-usuario/carlo-acutis.git
cd carlo-acutis/CarloAcutis
dotnet run
```

Acesse `https://localhost:5001` no navegador.
